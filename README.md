# Jify Discount

> Threshold-based discount engine with scheduling and fee-based calculation for WooCommerce.

![PHP 7.4+](https://img.shields.io/badge/PHP-7.4%2B-777bb4)
![Stable](https://img.shields.io/badge/stable-2.1.0-brightgreen)

Create automatic, threshold-driven discount rules that apply as negative WooCommerce fees. Unlike standard coupons, these activate on cart subtotals and can be scheduled in advance.

## Key Features

- **Threshold rules** — e.g. "Spend $1000, get $100 off"
- **Fixed amount or percentage** — both calculation modes
- **Scheduled promotions** — start / end date per rule
- **Marketing messages** — upsell hints in the cart totals area
- **Fee-based logic** — plays nicely with tax and shipping calculations
- **Smart sorting** — discounts appear in the correct order relative to taxes

## Installation

1. Upload to `/wp-content/plugins/jify-discount`.
2. Activate from the **Plugins** screen.
3. Configure under **Product Data → Jify Discount**.
